II. Hướng dẫn biên dịch và chạy mã nguồn
* Tên bài tập: Ứng dụng generic và collections để tạo ứng dụng từ điển
* Cấu trúc source code:
- Package:
  + Model: Chứa các class cho việc lưu từ vựng
  + Data: Chứa các file dữ liệu của chương trình gồm 2 file chứa từ vựng và 2 file lưu dữ liệu khi chương trình thực   thi
  + Main: Chứa các file giao diện và logo của chương trình. Trong đó file MainScreen.java là file đầu vào.

*Thư viện:
  + jcalendar: Dùng để tạo ra UI cho phếp người dùng nhập ngày tháng năm

* Cách biên dịch và chạy chương trình:
- Ta mở project bằng IDE (ưu tiên chọn NetBeans 17 vì project được code trên NetBeans)
- Để chạy chương trình ta mở package main và chọn file MainScreen.java và tiến hành Run file (Shift + F6).

* Chú ý: Khi thực hiện TRA TỪ ĐIỂN, THÊM TỪ, THÊM TỪ ƯA THÍCH, XÓA TỪ cần phải SAVE* dữ liệu. Vì dữ liệu từ điển khá lớn nên mỗi thực hiện thao tác nếu đều lưu lại thì chương trình sẽ có hiệu suất thấp nên sẽ có nút SAVE* nhằm chỉ lưu 1 lần cho mọi thao tác.
